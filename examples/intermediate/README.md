# Intermediate Examples

Real-world examples for productive development workflows.

## What's Here

Framework-specific and project-oriented examples:
- React/Vue/Angular skills
- Python/Node.js/Java helpers
- Database management tools
- API development aids
- Testing workflows

## Available Examples

### Framework Skills (`framework-skills/`)
- **React Component Generator**: Creates React components with TypeScript and best practices
- **Django Model Helper**: Generates Django models with relationships and migrations

### Workflow Commands (`workflow-commands/`)
- **PR Creator**: Creates pull requests with auto-generated descriptions
- **Deploy Script**: Deploys applications with safety checks and verification

### Advanced Hooks (`advanced-hooks/`)
- **Pre-Commit Lint**: Runs linting before allowing commits
- **Test Enforcement**: Ensures tests exist and pass before key operations

### Domain Subagents (`domain-subagents/`)
- **API Designer**: Designs RESTful APIs with proper conventions
- **Database Architect**: Creates database schemas with normalization and indexes

### MCP Integrations (`mcp-integrations/`)
- **GitHub Workflows**: Integrates GitHub Actions for CI/CD automation
- **Database Connections**: Connects to databases for queries and analysis
