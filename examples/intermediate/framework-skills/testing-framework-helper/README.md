# Testing Framework Helper

Generates comprehensive test suites with unit, integration, and E2E tests for various frameworks.

## Installation

```bash
cp examples/intermediate/framework-skills/testing-framework-helper/SKILL.md \
   .claude/skills/testing-framework-helper.md
```

## Usage

```
"write tests for UserService"
"create test suite for authentication"
"generate E2E tests for checkout flow"
```

## Features

- ✅ Multiple frameworks (Jest, Vitest, pytest, Mocha)
- ✅ Unit tests with mocking
- ✅ Integration tests
- ✅ E2E tests (Playwright, Cypress)
- ✅ Test fixtures and factories
- ✅ Coverage setup
- ✅ Assertions and matchers
- ✅ Async test handling
