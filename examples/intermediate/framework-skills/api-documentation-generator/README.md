# API Documentation Generator

Creates comprehensive OpenAPI/Swagger specs and API documentation.

## Installation

```bash
cp examples/intermediate/framework-skills/api-documentation-generator/SKILL.md \
   .claude/skills/api-documentation-generator.md
```

## Usage

```
"generate OpenAPI spec for my REST API"
"create API documentation for user endpoints"
"build Swagger docs for product API"
```

## Features

- ✅ OpenAPI 3.0 specification
- ✅ Detailed endpoint documentation
- ✅ Request/response schemas
- ✅ Authentication schemes
- ✅ Examples and descriptions
- ✅ Error response documentation
- ✅ Swagger UI compatible
