# GraphQL Schema Generator

Creates GraphQL schemas with types, queries, mutations, resolvers, and DataLoaders.

## Installation

```bash
cp examples/intermediate/framework-skills/graphql-schema-generator/SKILL.md \
   .claude/skills/graphql-schema-generator.md
```

## Usage

```
"create GraphQL schema for e-commerce"
"generate GraphQL API for blog with posts and comments"
"build user management GraphQL schema"
```

## Features

- ✅ Type definitions
- ✅ Queries and mutations
- ✅ Resolvers with proper structure
- ✅ DataLoader for N+1 prevention
- ✅ Input types and validation
- ✅ Subscriptions support
- ✅ Error handling
- ✅ Authentication/authorization
