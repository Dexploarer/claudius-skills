# Next.js Page Generator

Generates Next.js 13+ pages with App Router, Server/Client Components, and modern data fetching.

## Installation

```bash
cp examples/intermediate/framework-skills/nextjs-page-generator/SKILL.md \
   .claude/skills/nextjs-page-generator.md
```

## Usage

```
"create Next.js page for user dashboard"
"generate product listing page with SSR"
"build blog post page with static generation"
```

## Features

- ✅ App Router (Next.js 13+)
- ✅ Server and Client Components
- ✅ Server-side rendering (SSR)
- ✅ Static site generation (SSG)
- ✅ Data fetching patterns
- ✅ Loading and error states
- ✅ Metadata API
- ✅ TypeScript support
