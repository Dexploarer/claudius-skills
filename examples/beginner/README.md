# Beginner Examples

Simple, focused examples to help you understand each pillar of Claude Code.

## What's Here

This directory contains beginner-friendly examples that demonstrate:
- Single-purpose skills
- Simple slash commands
- Basic hooks
- Introductory subagents
- Basic MCP configurations
