```
[Show impressive gameplay moment]

[STEP 1 - 0:45-1:30]

[Show prompt clearly on screen]
[Highlight key parts]
"Notice how we specify exactly what we want..."

[STEP 2 - 1:30-2:15]
"Step 2: AI generates the complete game code. Look at this:"
[Show code being generated]
[Point out key sections]
"500 lines of code in 30 seconds. That's the power of AI."

[STEP 3 - 2:15-3:00]
"Step 3: Customize it. Just tell AI what you want different:"
[Show customization prompts]
[Show changes happening live]
"Want it cyberpunk theme? Done. Different enemies? Easy."

[RESULT - 3:00-3:30]
"And there you have it - a complete game in under 3 minutes."
[Show final game]
"Now it's your turn. Download the prompt in the description,
try it yourself, and share what you build in the community!"

[CALL TO ACTION - 3:30-3:45]
"Click the link below to get started. Join our community where
500+ game developers are building with AI. See you inside!"

[END SCREEN]
- Link to course
- Join community button
- Subscribe/Follow
```

**Recording Notes:**
- Record screen + face (picture-in-picture)
- Use clear, energetic voice
- Show, don't just tell
- Keep pace fast (no dead air)
- Music: Upbeat, modern (royalty-free)

## Recording Best Practices

### Equipment

**Minimum Setup:**
- Decent microphone ($50+)
- Screen recording software (OBS, Loom)
- Quiet room
- Good lighting (for face cam)

**Optimal Setup:**
- Quality USB mic ($100-200)
- Ring light
- Dual monitors (one for recording, one for script)
- Video editing software (DaVinci Resolve - free!)

### Recording Tips

**Before Recording:**
- [ ] Write script
- [ ] Practice once
- [ ] Close unnecessary programs
- [ ] Set up screen resolution (1920x1080)
- [ ] Test audio levels
- [ ] Check lighting
- [ ] Have water nearby

**During Recording:**
- Speak clearly and enthusiastically
- Vary your tone (no monotone!)
- Pause between sections (easier editing)
- If you make mistake, pause 3 seconds and restart sentence
- Smile (it shows in your voice!)
- Point at things on screen
- Use your hands (if on camera)

**After Recording:**
- Watch full video once
- Note timestamps for cuts
- Remove long pauses
- Add music (low volume)
- Add text overlays for key points
- Export in 1080p
- Create thumbnail

### Editing Checklist

**Must Have:**
- [ ] Intro (0-5 seconds)
- [ ] Background music (subtle)
- [ ] Remove long pauses
- [ ] Add chapter markers
- [ ] Include captions/subtitles
- [ ] End screen (last 20 seconds)
- [ ] Thumbnail created

**Nice to Have:**
- [ ] B-roll footage
- [ ] Zoom effects on important parts
- [ ] Animated text overlays
- [ ] Sound effects (subtle!)
- [ ] Transitions between sections
- [ ] Graphics/diagrams

## Batch Recording Strategy

**Record Multiple Videos in One Session:**

1. Set up once
2. Record 4-6 short videos (using templates)
3. Edit all at once
4. Schedule across 2-3 weeks

**Benefits:**
- Consistent background/lighting
- Maintain energy
- Efficient use of time
- Always have content ready
