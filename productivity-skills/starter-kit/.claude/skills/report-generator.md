# Professional Report Generator

**Trigger phrases:**
- "create a report about"
- "generate report for"
- "write a report on"
- "summarize this into a report"
- "format this as a report"
- "executive report"

**Description:**
Generates professional, well-structured reports from raw data, notes, or high-level topics. Automatically formats with executive summaries, clear sections, data visualization suggestions, and actionable recommendations.

## What this skill does:

When you provide information or a topic, this skill will:

1. **Structure the report professionally:**
   - Executive Summary
   - Introduction/Background
   - Methodology (if applicable)
   - Findings/Analysis
   - Recommendations
   - Conclusion
   - Appendices (if needed)

2. **Format for clarity:**
   - Clear headings and subheadings
   - Bullet points and numbered lists
   - Tables for data comparison
   - Callout boxes for key insights
   - Professional tone and language

3. **Add strategic elements:**
   - Key takeaways highlighted
   - Data visualization suggestions
   - Action items with owners
   - Next steps clearly defined

4. **Tailor to audience:**
   - Executive level (high-level, strategic)
   - Management (balanced detail and strategy)
   - Technical (detailed, data-driven)
   - Board/investors (results and projections)

## Best Practices:

1. **Always include an executive summary:**
   - Busy executives may only read this
   - Include key findings and recommendations
   - Keep to one page

2. **Use data visualization:**
   - Tables for comparisons
   - Charts/graphs for trends
   - Callout boxes for key metrics

3. **Be action-oriented:**
   - Clear recommendations
   - Next steps defined
   - Owners assigned

4. **Maintain professional tone:**
   - Objective and factual
   - Support claims with data
   - Avoid jargon (or define it)

## Tips for better reports:

- "Create an executive report about..." (shorter, high-level)
- "Generate a detailed technical report on..." (in-depth, data-heavy)
- "Write a status report for stakeholders..." (specify audience)
