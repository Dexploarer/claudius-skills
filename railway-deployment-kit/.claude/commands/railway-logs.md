# Railway Logs Viewer

View and analyze logs from Railway services with filtering and search capabilities.

## Tasks

1. **Service Selection**
   - List all services in the project
   - Allow user to select specific service or view all
   - Show available log sources

2. **Log Retrieval**
   - Fetch recent logs using `railway logs`
   - Support follow mode for real-time logs
   - Handle multiple services if requested

3. **Log Analysis**
   - Identify errors and warnings
   - Detect common issues:
     - Connection failures
     - Build errors
     - Runtime exceptions
     - Performance issues
   - Highlight critical log entries

4. **Filtering Options**
   - Filter by log level (error, warn, info, debug)
   - Search for specific patterns
   - Time range filtering
   - Service-specific filtering

5. **Actionable Insights**
   - Suggest fixes for common errors
   - Provide debugging commands
   - Link to relevant documentation
   - Recommend monitoring tools

## Expected Output

- Formatted log output
- Error and warning highlights
- Issue summary and analysis
- Debugging suggestions
- Commands for deeper investigation
